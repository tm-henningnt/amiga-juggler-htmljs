About

Eric Graham's 1987 Amiga Juggler source was recovered and published in AlphaPixel's GitHub archive. This rebuild uses the original static .dat scenes for geometry, cameras, materials, and lamps. The juggler motion is reconstructed from historical movie output, Ernie Wright's notes, and Meatfighter's Java reimplementation because the scene files contain no reusable animation tracks.
